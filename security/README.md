# Security & Pentesting Tools

A curated, browser-importable directory of 71 security, penetration-testing, and OSINT resources — from reconnaissance and exploitation to forensics, threat intel, and hands-on practice.

Import `security.html` into your browser the same way as the [OSINT collection](../osint/README.md#method-to-import-bookmark).

> For lawful, authorized security testing and education only.
